Critical Error: UrlReWriteProvider

Specific Error Message: [UrlReWriteProvider]

Please contact your site administrator. An explanation for why this error could occur is below.

Zimbra Community expects to find a type which inherits from UrlReWriteProvider configured in the communityserver.config file's provider section. Once the type is correctly added, you will need to do an application restart to reload the providers.

For more information you can also visit http://zimbra.com.
