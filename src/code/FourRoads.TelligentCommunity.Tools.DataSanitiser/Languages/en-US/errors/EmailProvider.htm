Critical Error: EmailProvider

The dataprovider class "[EMAILPROVIDERCLASS]" could not be loaded.

Please contact your Zimbra Community administrator. A technical explanation of why this error is caused is below.
Zimbra Community expects to load a email provider class; in this case: [EMAILPROVIDERCLASS].

This class requires that certain methods and properties are implemented.
